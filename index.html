ALX Software Engineering Final Project

3D Ray-Casting Game

Introduction:

Welcome to my final project for ALX Software Engineering! This 3D ray-casting game is developed using SDL2 and C, chosen for their low-level access to hardware and graphics, as well as for learning purposes regarding game development and ray tracing.

Features:

  • Player Movement: Implementing player movement and collision detection with walls.
  • Ray Casting: Using ray casting to project a 3D world onto a 2D projection plane.

Challenges:

  • Adding additional characters.
  • Deciding on map layout.

About:

This project was inspired by a passion for game development and a desire to learn low-level programming with SDL2 and C. It is a Portfolio Project for Holberton School. You can find more about Holberton School here.
